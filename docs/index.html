About the Project

Habitat is a mobile habit-tracking application that helps users build better routines. Users can create habits, maintain streaks, and view friends’ public habits to stay motivated.

This project demonstrates a full-stack architecture including:

  • Mobile client built with React Native and Expo
  • Backend server built with Node.js and Express
  • Database powered by PostgreSQL and Prisma

Android Installation

Please install the following on your Android device:

Download Habitat on Android

Otherwise, you may paste this link into your Android device's browser and tap "Install":

https://expo.dev/accounts/swethavanith/projects/mobile/builds/4f539ff2-a2ec-4ab9-96ec-39e9c4c03571

Steps

  1. Tap the link above and then tap "Download".
  2. Download .apk.
  3. Launch and open Android emulator.
  4. Drag the downloaded .apk file into the emulator.
  5. The app will be installed and can be opened from the emulator.

iOS Installation

Please install the following on your iOS device:

Download Habitat on iOS

Otherwise, you may paste this link into your iOS device's browser and tap "Install":

https://expo.dev/accounts/swethavanith/projects/mobile/builds/290a6fb0-80af-4836-b4b5-a6639081e0c9

Steps

  1. Tap the link above and then tap "Download".
  2. Download .app.
  3. Launch and open iOS simulator.
  4. Install the app using this command, replacing the path as necessary:
xcrun simctl install booted ~/Downloads/MyApp.app
  1. Find the bundle identifier using this command:
plutil -p ~/Downloads/MyApp.app/Info.plist | grep CFBundleIdentifier
  1. Launch the app using this command, replacing the bundle identifier:
xcrun simctl launch booted your.bundle.identifier
  1. The app will be installed and can be opened from the simulator.

Source Distribution: Running the Project Locally

Before running the setup script, ensure the following are installed on your machine:

  • Node.js v18 or higher
  • npm
  • Expo Go app, on your mobile device or through your choice of emulator

To verify installation, run node -v and npm -v in your terminal. If these commands return a version number, you are ready to proceed.

Start in the root directory of our project found in the repository.

Source Code

View the complete project repository on GitHub:

View GitHub Repository

Run the following to clone the repository:

git clone https://github.com/sanikadeshmukh3/Habitat.git
cd Habitat

Start the backend server and mobile application

Please make sure your emulator is already running.

In one terminal window, please run the following commands:

./setup.sh

Then run the app on the emulator by entering a for Android or i for iOS when Expo prompts.

The login feature connects to the backend server and validates credentials using the PostgreSQL database, demonstrating a complete round-trip data operation.
