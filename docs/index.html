About the Project

Habitat is a mobile habit-tracking application that helps users build better routines. Users can create habits, maintain streaks, and view friends’ public habits to stay motivated.

This project demonstrates a full-stack architecture including:

  • Mobile client built with React Native and Expo
  • Backend server built with Node.js and Express
  • Database powered by PostgreSQL

Start in the root directory of our project found in the repository.

Note: The current code supports the Android emulator, but in order to run the app on iOS, you will need to have Xcode installed. The ip address for the backend server will need to be updated accordingly. To do so, open the mobile/app/login.tsx file and update the fetch URL to your computer's IP address.

For iOS:http://localhost:3000/login

For Android:http://10.0.2.2:3000/login

1. Start the backend server

In one terminal window, please run the following commands:

cd backend
npm install
npm run dev

2. Start the mobile application

Please make sure your emulator is already running.

In a new terminal window, please run the following commands:

cd mobile
npm install
npx expo start

Then run the app on the emulator by entering a for android or i for iOS when expo prompts.

The login feature connects to the backend server and validates credentials using the PostgreSQL database, demonstrating a complete round-trip data operation.
